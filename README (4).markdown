## 🌈 Featured Projects

### [🎮 Virtual Mouse](https://github.com/stebyvarghese1/virtualmouse)
Wave your hand, own the screen! This project uses webcam-based gesture recognition to control your PC with clicks, scrolls, drags, and zooms. It’s a vibe for accessibility and futuristic control. 🤲

- **Tech**: Python 🐍 | MediaPipe 🤲
- **Features**: Real-time hand tracking ⏱️ | Left/Right/Double-click 🖱️ | Pinch-to-zoom 🔍 | Drag mode ✋
- **License**: MIT 📜

### [🛡️ SOC Platform](https://github.com/stebyvarghese1/soc_platform-)
A cybersecurity banger! This AI-powered Security Operations Center detects threats in real-time, pulls logs from multiple systems, and generates clean PDF reports. Protecting the digital world with style. 🔒

- **Tech**: Flask 🍶 | MongoDB 🍃 | Sentence-Transformers 🤖 | ReportLab 📊 | Matplotlib 📈
- **Features**: AI-driven threat detection 🚨 | WebSocket alerts 📡 | Cross-platform log collection 🌐
- **Status**: Ready to launch 🚀

### [📱 CamDroid](https://github.com/stebyvarghese1/Camdroid)
Turn your phone into a wireless webcam with a quick QR scan. Stream video over Wi-Fi with a smooth, secure, and vibey interface. No cables, just pure tech magic! 📷

- **Tech**: Python 🐍 | Kivy 📱 | PyQt5 🖥️ | Flask 🍶 | OpenCV 📷
- **Features**: QR-secured streaming 🔐 | Camera toggle 🔄 | Mirror mode 🪞 | Dark-mode UI 🌙
- **License**: MIT 📜

### [🌳 Memory Tree](https://github.com/stebyvarghese1/Memory_tree)
A chill project visualizing memory as tree structures. It’s my creative space for building a dope data visualization tool—stay tuned for the glow-up! 🌱

- **Tech**: Python 🐍
- **Status**: Work in progress ⏳
