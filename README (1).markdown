## 🌈 Featured Projects

### [🎮 AI Virtual Mouse](https://github.com/stebyvarghese1/virtualmouse)
Control your PC with a wave of your hand! This vibe-packed project uses webcam-based gesture recognition to handle clicks, scrolls, drags, and zooms. Perfect for accessibility and next-level demos. 🤲

- **Tech**: Python 🐍 | MediaPipe 🤲
- **Features**: Real-time hand tracking ⏱️ | Left/Right/Double-click 🖱️ | Pinch-to-zoom 🔍 | Drag mode ✋
- **License**: MIT 📜

### [🛡️ SOC Platform](https://github.com/stebyvarghese1/soc_platform-)
A cybersecurity beast! This AI-driven Security Operations Center detects threats in real-time, aggregates logs, and generates sleek PDF reports. Built to keep the digital world safe with style. 🔒

- **Tech**: Flask 🍶 | MongoDB 🍃 | Sentence-Transformers 🤖 | ReportLab 📊 | Matplotlib 📈
- **Features**: AI-powered threat detection 🚨 | WebSocket alerts 📡 | Cross-platform log collection 🌐
- **Status**: Ready to deploy 🚀

### [📱 CamDroid](https://github.com/stebyvarghese1/Camdroid)
Turn your phone into a wireless webcam with a quick QR scan. Stream video over Wi-Fi with a smooth, secure, and vibey interface. No cables, just pure innovation! 📷

- **Tech**: Python 🐍 | Kivy 📱 | PyQt5 🖥️ | Flask 🍶 | OpenCV 📷
- **Features**: QR-secured streaming 🔐 | Camera toggle 🔄 | Mirror mode 🪞 | Dark-mode UI 🌙
- **License**: MIT 📜

### [🌳 Memory Tree](https://github.com/stebyvarghese1/Memory_tree)
A creative vibe in progress! This project visualizes memory as tree structures, set to become a dope data visualization tool. Stay tuned for the magic! 🌱

- **Tech**: Python 🐍
- **Status**: Work in progress ⏳
