## 🌈 My Top Projects

### [🎮 AI Virtual Mouse](https://github.com/stebyvarghese1/virtualmouse)
Wave your hand, control your PC! This project uses webcam-based gesture recognition to let you click, scroll, drag, and zoom with just your fingers. It’s like magic for accessibility and fun! 🤲

- **Tech**: Python 🐍 | MediaPipe 🤲
- **Features**: Real-time hand tracking ⏱️ | Left/Right/Double-click 🖱️ | Pinch-to-zoom 🔍 | Drag mode ✋
- **License**: MIT 📜

### [🛡️ SOC Platform](https://github.com/stebyvarghese1/soc_platform-)
A next-level cybersecurity tool! This AI-powered Security Operations Center spots threats in real-time, pulls logs from multiple systems, and creates slick PDF reports. Keeping the digital world safe! 🔒

- **Tech**: Flask 🍶 | MongoDB 🍃 | Sentence-Transformers 🤖 | ReportLab 📊 | Matplotlib 📈
- **Features**: AI-driven threat detection 🚨 | WebSocket alerts 📡 | Cross-platform log collection 🌐
- **Status**: Ready to roll 🚀

### [📱 CamDroid](https://github.com/stebyvarghese1/Camdroid)
Turn your phone into a wireless webcam with a quick QR scan! Stream video over Wi-Fi with a clean, secure, and user-friendly interface. No cables, just pure innovation. 📷

- **Tech**: Python 🐍 | Kivy 📱 | PyQt5 🖥️ | Flask 🍶 | OpenCV 📷
- **Features**: QR-secured streaming 🔐 | Camera toggle 🔄 | Mirror mode 🪞 | Dark-mode UI 🌙
- **License**: MIT 📜

### [🌳 Memory Tree](https://github.com/stebyvarghese1/Memory_tree)
A cool project exploring memory visualization as tree structures. It’s my playground for creating something epic in data visualization—stay tuned! 🌱

- **Tech**: Python 🐍
- **Status**: Work in progress ⏳
